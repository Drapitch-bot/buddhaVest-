# BuddhaVest - האתר (Frontend)

## מה זה
קובץ HTML אחד שמציג את הניתוח בעיצוב הכהה והצבעוני שראינו - חיפוש מנייה,
עיגול ההחלטה, כל המדדים, דיבידנד, וגרף מחיר.

הקובץ הזה **לא עובד לבד** - הוא צריך לדבר עם שרת ה-backend שכבר בנינו
(אותו שרת שראית ב-http://127.0.0.1:8000).

## איך מריצים

1. ודא שה-backend רץ:
   - לחיצה כפולה על `run_buddhavest.bat` (מתיקיית ה-backend)
   - תשאיר את חלון ה-CMD הזה פתוח ברקע

2. עכשיו, מתיקיית ה-frontend, לחיצה כפולה על `index.html`
   - זה יפתח בדפדפן כדף רגיל

3. בתיבת החיפוש, תכתוב AAPL ותלחץ Enter (או תלחץ על אחד מהכפתורים המהירים)

## שימוש בטלפון (מאוחר יותר)
כרגע הדף עובד רק במחשב, כי הוא מדבר עם שרת שרץ עליו (127.0.0.1 = "המחשב הזה").
כדי שזה יעבוד גם בטלפון, נצטרך להעלות את ה-backend לאינטרנט (Hosting חינמי) -
זה השלב הבא לאחר שנאשר שהכל עובד מקומית.

## אם משהו לא עובד
- מסך אדום "Couldn't reach BuddhaVest's server" -> ה-backend לא רץ. הרץ את run_buddhavest.bat.
- מסך אדום אחר -> תעתיק לי את הטקסט שמופיע ונבדוק.
